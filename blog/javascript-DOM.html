JAVASCRIPT & DOM EXPLORATION

Lets talk analogies re. HTML and CSS:

HTML and CSS creates a house. Let me explain how:

HTML is the frame of the house: how many floors the house will have, where the walls will be, and which room you will put furniture/appliances. Likewise, for a website, HTML sets out code sections (walls/storeys) and which sections will house which elements including text and images (furniture/appliances).

CSS would then be the specific design and positioning of everything: the type of wallpaper or paint/color of the flooring, how big the rooms are, where exactly in the room the furniture/applicances is/are positioned. Same goes for a website, CSS sets the colors of the page or elements (walls/floors/rooms), how big elements will be (floor/room sizes) and where in the element will text and images be (furniture and appliance positioning).

As an extra to this anaolgy JavaScript can be seen as the functionality of the house: the doorbell ringing when pressed, turning the TV from the input from the remote and starting the dishwasher after dinner. Likewise, JavaScript can hook onto events from the user (clicking doorbell & remote button clicking) and provide certain functionality or feedback (doorbell ringing & TV showing you the news).
